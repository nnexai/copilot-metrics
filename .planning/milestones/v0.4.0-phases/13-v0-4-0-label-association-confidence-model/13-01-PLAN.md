# Phase 13 Plan: v0.4.0 label association confidence model

## Goal

Preserve granular label association evidence and compute deterministic per-session label confidence rankings without making final scores the only stored truth.

## Tasks

1. Add a versioned label confidence scorer that derives rankings from `label_evidence` rows.
2. Keep scoring algorithm-time/query-time so future algorithms can reuse the same evidence.
3. Weight branch/cwd/folder evidence very highly, metadata/context evidence highly, and prompt/response/tool mentions lower but accumulating.
4. Deduplicate identical evidence during scoring and skip duplicate evidence inserts.
5. Add deterministic tie-breaking across equal scores.
6. Expose machine-readable confidence summaries in existing JSON label report payloads without changing human report totals.
7. Add tests for dominance, accumulation, duplicate de-dupe, tie-breaking, and JSON-visible confidence metadata.
